# 🍽️ FlavourNest - Food Recipe App

A full-stack food recipe sharing platform built with React and Node.js.

## 🚀 Tech Stack

- **Frontend:** React, Vite, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Auth:** JWT, Bcrypt

## 📁 Project Structure

Flavournest_Recipe/
├── flavournest-backend/ # Node.js REST API
└── flavournest-frontend/ # React Application

## ⚙️ Setup Instructions

### Backend

```bash
cd flavournest-backend
npm install
# Create .env file (see .env.example)
npm run dev
```

### Frontend

```bash
cd flavournest-frontend
npm install
npm run dev
```
